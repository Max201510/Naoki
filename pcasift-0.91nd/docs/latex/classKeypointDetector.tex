\subsection{Detailed Description}
\doxyref{Keypoint}{p.}{classKeypoint} detector class. 

Finds keypoints using David Lowe\textquotesingle{}s S\+I\+FT algorithm. Computes local descriptors using P\+C\+A\+S\+I\+FT.

Limiations\+: Does not fully implement Lowe\textquotesingle{}s S\+I\+FT algorithm. Does not interpolate in location and scale.
